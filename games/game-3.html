Uses xAI Responses API with web_search + x_search (X uses a UTC date window for the last ~12h). Then five short articles from what was actually retrieved. If your key blocks tools, the server falls back to plain chat (not live). Env: XAI_NEWSPAPER_CHAT_ONLY=1 skips search; XAI_NEWSPAPER_X_ONLY=1 / XAI_NEWSPAPER_WEB_ONLY=1 limit tools.

Needs XAI_API_KEY. For live news, allow Responses (/v1/responses), Web Search, and X Search on the key, plus your model. XAI_MODEL pins the model id. The line above shows API build after each request so you can confirm the deploy.
