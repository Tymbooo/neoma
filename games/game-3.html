The tech desk

Uses xAI Responses API with x_search (X posts in a UTC date window covering the last ~12h), then five short articles. If your key blocks that, the server falls back to plain chat (no live X). Optional env: XAI_NEWSPAPER_CHAT_ONLY=1 to skip search.

Needs XAI_API_KEY. For live X, allow Responses (/v1/responses) and X Search on the key, plus your model. XAI_MODEL pins the model id.
