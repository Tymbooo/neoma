The tech desk

Live web and X search for tech news (last ~12 hours; X uses a UTC date window), then five short articles via xAI Responses. If your key blocks tools, the app falls back to plain chat (not live).

Needs XAI_API_KEY. For live results: enable Responses, Web Search, and X Search on the key, plus your model. Optional XAI_MODEL pins the model id.
