The tech desk

Sends a short user prompt: five biggest tech stories from the last 12 hours, as five concise articles (current UTC is included). Plain chat has no live web or X feed unless you add search tools on the API side.

Needs XAI_API_KEY (Vercel or .env.local). If the key uses Restrict access, allow Chat and tick the exact model you use, or set XAI_MODEL to that id (e.g. grok-4-1-fast-reasoning).
